# Planned Tables

**Purpose:** Brief descriptions of core NexLead database tables.

- **users** — Authentication identity and profile data.
- **workspaces** — Agency/account container for all operational data.
- **workspace_members** — User-to-workspace membership and roles.
- **leads** — Prospective client records and pipeline status.
- **lead_audits** — Website audit runs linked to leads.
- **audit_issues** — Individual findings from an audit.
- **outreach_campaigns** — Grouped outbound initiatives.
- **outreach_messages** — Personalized messages per lead.
- **pipeline_stages** — Configurable pipeline columns per workspace.
- **meetings** — Scheduled calls and prep context.
- **subscriptions** — Billing plan state per workspace.
