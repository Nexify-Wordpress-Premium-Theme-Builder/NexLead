# Supabase

Supabase projesi için migration, seed, RLS policy ve edge function dosyaları.

| Klasör | Açıklama |
|--------|----------|
| `migrations/` | SQL migration dosyaları |
| `seed/` | Geliştirme ortamı seed verileri |
| `policies/` | Row Level Security politikaları |
| `functions/` | Supabase Edge Functions |
