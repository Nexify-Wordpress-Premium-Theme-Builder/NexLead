# @nexlead/web

NexLead frontend uygulaması — Next.js App Router.

## Klasör Yapısı

| Klasör | Açıklama |
|--------|----------|
| `app/` | Sayfa rotaları ve layout'lar |
| `components/` | Sunum katmanı bileşenleri |
| `features/` | Modül bazlı iş mantığı ve bileşenler |
| `hooks/` | React hook'ları |
| `lib/` | API client ve harici servis bağlantıları |
| `providers/` | Context provider'lar |
| `styles/` | Global stiller ve tema |
| `types/` | Uygulamaya özel tipler |
| `utils/` | Yardımcı fonksiyonlar |

## Geliştirme

```bash
pnpm --filter @nexlead/web dev
```
