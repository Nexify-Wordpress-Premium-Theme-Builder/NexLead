# @nexlead/ui

Paylaşılan UI bileşen kütüphanesi — design token'lar, primitive'ler ve bileşenler.

## Klasör Yapısı

| Klasör | Açıklama |
|--------|----------|
| `src/components/` | Bileşik UI bileşenleri |
| `src/primitives/` | Temel yapı taşları (Button, Input, vb.) |
| `src/icons/` | İkon seti |
| `src/tokens/` | Renk, tipografi ve spacing token'ları |

Light / premium / clean tasarım sistemi burada tanımlanacak.
